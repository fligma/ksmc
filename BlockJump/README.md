# BlockJump - Geometry Dash Clone for Terminal

A simple Geometry Dash-like game built with KSMC (KSM Compiler/Interpreter) that runs in your terminal!

## How to Play

1. **Run the game:**
   ```bash
   python ksmc.py BlockJump/blockjump.ksm
   ```

2. **Controls:**
   - `SPACE` or `W` - Jump
   - Avoid the obstacles (`#`) by jumping over them

3. **Objective:**
   - Reach the end of the level
   - Don't hit the obstacles or it's game over!
   - Your score is based on how far you progress
